# 원티드 프론트엔드 프리온보딩 인턴십 선발과제

지원자의 성명: 강석규

프로젝트의 실행 방법
- `git clone https://github.com/AlgeMoya/wanted-pre-onboarding-frontend.git`
- `cd wanted-pre-onboarding-frontend`
- `npm install`
- `npm start`
